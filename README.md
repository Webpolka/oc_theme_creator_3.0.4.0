# Создатель тем для OpenCart версии 3.0.4.0 2025
Создатель тем для OpenCart версии 3.0.4.0 до:

## Как это работает
Создает расширение: копирует файлы шаблонов из "./files" в новую временную папку, заменяет содержимое файла и имена файлов:
- %name% на $name;
просто преренесите сгенерированные папки admin и catalog поверх существующих и сгенерированная тема добавиться ! Готово !

## Как это использовать
php ./theme-creator.php [имя] 
 имя - название темы (префиксы Theme или theme_ темы не указываются).
 
 генератор здесь же создаст готовую папку с начальными файлами готовыми к использованию
