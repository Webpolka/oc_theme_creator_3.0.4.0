# Генератор стартовой темы для OpenCart версии 3.0.4.0 в 2025

Откройте терминал и запустите php код ! Укажите название темы ! 
Будет сгенерирована начальная тема для разработки в ocmod.zip формате готовая к установке !

## Как это работает
Создает расширение: копирует файлы шаблонов из "./files" в новую временную папку, заменяет содержимое файла и имена файлов:
- %name% на $name;
просто установите тему и она сразу примениться, так как в ней уже настроено автоматическое активирование !

## Как это использовать ?
Откройо терминал в своем VSCode и выполни php скрипт:

php ./theme-create.php [имя] 
имя - название темы (префиксы Theme или theme_ темы не указываются).
